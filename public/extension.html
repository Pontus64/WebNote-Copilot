悬浮笔记 · 浏览器扩展

在任意网页右下角加上悬浮笔记 + AI 聊天面板,并解除 GitHub 等站点对内嵌面板的安全策略限制。装一次,处处可用。

下载扩展(.zip)
支持 Chrome / Edge / Brave 等 Chromium 内核浏览器

安装步骤

  1. 下载上面的 floating-notes-extension.zip,并解压成一个文件夹。
  2. 在浏览器地址栏打开 chrome://extensions(Edge 用 edge://extensions)。
  3. 打开页面右上角的「开发者模式」开关。
  4. 点击「加载已解压的扩展程序」,选择第 1 步解压出来的文件夹。
  5. 完成 ✅ 打开任意网页,右下角会出现悬浮按钮。
关于开发者模式提示: Chrome 每次启动可能提示「停用开发者模式扩展」,点「保留」即可继续使用。 若某些站点(如网银)你希望保留其原有安全策略,点工具栏的扩展图标 → 「加入黑名单」即可对该站点关闭。
